    A List of my software projects

    I built these from scratch, at hackathons, for fun or in as school projects. Most were built in a small team.
    • Crash Alert (2018) - Android app that signals authorities when a user has been in a crash

    • Baltimore Safety Screen (2017) - Android app that shows safety levels and high risk crime hotspots to avoid in this city.

    • Offcampushousing (2017) - Web app I built to help me fine a sublet while I was not using the room in the summer. Ended up launching and have many other users.

    • MacArchives (2016) - Web app for university resources, has thousands of users now

    • Quick Question (2016) - Web app to allow students in a lecture hall to anonymously post QnA, profs see top question from class and gives students feedback.

    • TagMe (2015) - Android app built to share contact information and other social media connections by tapping phone together. Live an in the app store.

    • Fantasy Buddy (2015) - Android App built to track information around the web on Football players to assist Fantasy Football.

    • Recycle Rush (2014) - Find the closest Recycling/Garbage bin near you. Used Toronto Waste Management Database
